# SimpleMap

SimpleMap — это простая реализация интерфейса `Map` в Java, которая использует массив для хранения пар ключ-значение.
- `SimpleMap()`: Создает новый экземпляр `SimpleMap` с начальной емкостью 50 и коэффициентом загрузки 0.75.
Переменныые:
-size - переменная, отслеживающая количество элементов в массиве.
-table - массив, используемый для хранения пар ключ-значение.
-LOAD_FACTOR - константа, определяющая коэффициент загрузки (0.75). Если количество элементов в массиве превышает 75% от его емкости, то массив увеличивается в два раза.
-START_CAPACITY - константа, определяющая начальную ёмкость массива (50).

## Особенности:
- Использует массив для хранения пар ключ-значение.
- Реализованы методы для добавления, удаления, получения и проверки наличия элементов.
- Поддерживает ресайзинг массива при достижении определенного коэффициента загрузки.


### Основные методы:
size(): Возвращает текущее количество элементов в карте.
isEmpty(): Проверяет, пуста ли карта.
containsKey(Object key): Проверяет, содержит ли карта указанный ключ.
containsValue(Object value): Проверяет, содержит ли карта указанное значение.
get(Object key): Возвращает значение, связанное с указанным ключом.
put(K key, V value): Добавляет пару ключ-значение в карту.
remove(Object key): Удаляет пару ключ-значение, связанную с указанным ключом.
putAll(Map<? extends K, ? extends V> m): Добавляет все элементы из указанной карты в текущую карту.
clear(): Удаляет все элементы из карты.
keySet(): Возвращает множество всех ключей в карте.
values(): Возвращает коллекцию всех значений в карте.
entrySet(): Возвращает множество всех пар ключ-значение в карте.

### Вспомогательные методы:
- `private int getIndex(K key)`: Вычисляет индекс для ключа на основе его хэш-кода.
- `private void resize()`: Увеличивает размер массива в два раза, когда достигается порог заполнения.


Реализован внутреннй класс "Entry", который хранит пару ключ-значение.
Также, реализованы юнит-тесты с использованием библиотеки JUnit.
